# interlink-tv-tz

1. Установить логотип из папки logo при загрузке тв.
2. В углу лаунчера поставить также логотип вместо Vesta (левый верхний угол)
   
   <img width="212" height="113" alt="image" src="https://github.com/user-attachments/assets/04f337bc-a008-4906-abfb-b61b93a34150" />

4. удалить программы: WeTV, Visi Art
5. Отключить Рекомендации из лаунчера
   
   <img width="498" height="146" alt="image" src="https://github.com/user-attachments/assets/7e5a17c9-ac07-4b1e-8019-c89a6753ed5d" />

8. Установить программу TMS Player, ссылка: https://drive.google.com/file/d/1Vs5ne6EnHL-yCNIsuy4wMMKKhtw17Zz8/view?usp=sharing
9. Не можем дать права на удаленное управление клавиатурой, мышкой для программы RustDesk, как можно решить вопросy

11. Скрыть из нижней панели запуска программы: Netflix, Facebook, Prime_Video
