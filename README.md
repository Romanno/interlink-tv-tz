# interlink-tv-tz

1. Установить логотип из папки logo при загрузке тв.
2. В углу лаунчера поставить также логотип вместо Vesta (левый верхний угол)
3. удалить программы: WeTV, Visi Art
4. Отключить Рекомендации из лаунчера
5. Не можем дать права на удаленное управление клавиатурой, мышкой, как можно решить вопрос.
6. Установить программу TMS Player, ссылка: https://drive.google.com/file/d/1Vs5ne6EnHL-yCNIsuy4wMMKKhtw17Zz8/view?usp=sharing
7. на главной страницы Лаунчера закрепить в панели запуска программы первую программу: TMS Player
8. Скрыть из нижней панели запуска программы: Netflix, Facebook, Prime_Video
